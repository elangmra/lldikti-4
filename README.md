# Rebrand dan Redesign Website LLDIKTI 4

## About Project
This project aims to rebrand and redesign the LLDIKTI 4 website to appear more modern, fresh, and user-friendly. It focuses on improving UX/UI to ensure that the provided information is easily accessible and understandable by users.

## Features
- Modern design with intuitive UI
- Easy and responsive navigation for various devices
- Integration of search feature to facilitate information retrieval
- Enhanced accessibility to support users with diverse needs

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
