# Rebrand dan Redesign Website LLDIKTI 4
![FireShot Webpage Capture 003 - 'LLDIKTI Wilayah 4 - Lembaga Layanan Pendidikan Tinggi Wilayah 4' - www lldiktielang eu org](https://github.com/elangmra/LLDIKTI/assets/91105961/2f843686-7456-4431-9580-235e811494a8)

## Tentang Proyek
Proyek ini bertujuan untuk melakukan rebranding dan redesign website LLDIKTI 4 agar tampil lebih modern, fresh, dan user-friendly. Berfokus pada peningkatan UX/UI untuk memastikan bahwa informasi yang disediakan mudah diakses dan dipahami oleh pengguna.

## Fitur
- Desain modern dengan UI yang intuitif
- Navigasi yang mudah dan responsif untuk berbagai jenis perangkat
- Integrasi fitur pencarian untuk memudahkan pencarian informasi
- Aksesibilitas yang ditingkatkan untuk mendukung pengguna dengan berbagai kebutuhan

## Teknologi yang Digunakan
- HTML5
- CSS3
- JavaScript (ES6+)
